## JC8048S050N_I
This project uses capacitive touch by default. If the model is JC8048S050N_I, please change `#define ESP_PANEL_BOARD_USE_TOUCH (1)` to `#define ESP_PANEL_BOARD_USE_TOUCH (0)` in `demo/esp_panel_board_custom_conf.h`.
